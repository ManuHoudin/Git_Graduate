Ceci est un projet pour apprendre à ce servir de Git
L'exercice consiste à :
- créer un nouveau repository
- présenter le projet dans le fichier README.txt
- créer au moins 2 autres fichiers
- --> 2 fichiers index.html et style.css créés
- réaliser des modifications, des ajouts avec au moins 4 commits
- faire un push vers un nouveau repo sur GitHub
- envoyer l'URL du repo dans le fichier txt fourni
